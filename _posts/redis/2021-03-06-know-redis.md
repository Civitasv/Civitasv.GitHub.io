Redis，全称 Remote Dictionary Server（远程数据服务）。

Redis 整体的特点是速度快，支持的数据类型多。

`Redis vs Memcached`

- Redis 支持更多的数据类型，而且 Redis 数据库中的数据允许被持久化；
